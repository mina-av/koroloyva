Small uppercase kicker that sits above a headline to set section rhythm; carries a short leading rule by default.

```jsx
<Eyebrow>Ausgewählte Objekte</Eyebrow>
<Eyebrow rule={false} onDark>Services</Eyebrow>
```

Pair directly above an `h2`/display headline. Use `onDark` on ink sections; `rule={false}` for a bare label.
