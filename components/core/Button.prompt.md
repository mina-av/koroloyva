Primary action control — use for any call to action; squared edges, uppercase wide-tracked grotesk, a single brass accent.

```jsx
<Button variant="primary" size="lg" withArrow>Beratung anfragen</Button>
<Button variant="accent">Kurs sichern</Button>
<Button variant="secondary" onDark>Mehr erfahren</Button>
<Button variant="ghost" withArrow>Objekt ansehen</Button>
```

Variants: `primary` (ink fill), `accent` (brass), `secondary` (outline), `ghost` (underline link).
Sizes: `sm` / `md` / `lg`. Use `onDark` on ink sections; `withArrow` for editorial links; `fullWidth` in forms.
